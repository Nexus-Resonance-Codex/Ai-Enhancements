## Getting Started

### Installation

Clone the repository and install the dependencies into your environment:

```bash
git clone https://github.com/Nexus-Resonance-Codex/ai-enhancements.git
cd ai-enhancements
conda create -n nrc_ai python=3.10
conda activate nrc_ai
pip install -r requirements.txt
pip install -e .
```
